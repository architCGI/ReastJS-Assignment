# ReactJS Component Playground

A simple React application built to demonstrate core React concepts, including component-based architecture, client-side routing, navigation, and state management.

## Features

- React functional components
- React Router navigation
- Interactive counter component
- Reusable navigation bar
- Separate About and Contact pages
- Clean and responsive user interface

## Technologies Used

- React
- Vite
- React Router DOM
- JavaScript (ES6+)
- CSS3
